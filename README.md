# Codify (Modified from Code App)
## Aims
This project utilize the wasix abstraction to let ipad have the ability to execute arbitrary wasm code. like [nodejs](https://github.com/Multi-V-VM/node-wasix32), [rust](https://github.com/oligamiq/rust_wasm/) and [mysql](https://github.com/shota-kizawa/mysql-wasm). Now we are able to develop frontend backend on the iPad
## LLMs
This project integrate the CoreML to Vibe code on the iPad, correlated with the lsp and Codify highlighter.
