üì° API Endpoints

POST /api/marketplace/extensionquery
Search for extensions with filters and sorting
GET /api/marketplace/extensions/:publisher/:extension
Get detailed information about a specific extension
GET /api/marketplace/publishers/:publisher/vsextensions/:extension/:version/vspackage
Download extension .vsix package
GET /api/marketplace/featured?count=20
Get popular/featured extensions

üíª Usage Example

Search Extensions:

curl -X POST http://localhost:3000/api/marketplace/extensionquery \ -H "Content-Type: application/json" \ -d '{ "filters": [{ "criteria": [{"filterType": 8, "value": "python"}], "pageSize": 10 }], "flags": 914 }'

Download Extension:

curl -O http://localhost:3000/api/marketplace/publishers/ms-python/vsextensions/python/latest/vspackage

Use in Code App:

// Set in Extension Settings: Marketplace URL: http://localhost:3000/api/marketplace // Or use asplos.dev domain: Marketplace URL: https://asplos.dev/api/marketplace
